# MAPD-B Final Project: COVID19 Papers dataset analysis in a distributed environment
Final project of the MAPD-B course of the second semester of the first year of master degree in Physics of Data.

This project focuses on running a parallelized analysis through DASK of a dataset containing a large collection of COVID-19 related research papers.
The dataset comes from the **COVID-19 Open Research Dataset Challenge (CORD-19)**.

Different version of the dataset are available at: [CORD-19 on Kaggle](https://www.kaggle.com/allen-institute-for-ai/CORD-19-research-challenge) .<br> We decided to use the following version: **Version 95 (49 GB)** .
